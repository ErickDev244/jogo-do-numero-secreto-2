# 🔢 Numero Secret

O **Numero Secret** é um projeto simples desenvolvido em JavaScript que consiste em um jogo de adivinhação. O objetivo é descobrir o número secreto gerado aleatoriamente pelo sistema, recebendo dicas que ajudam o jogador a encontrar a resposta correta.

## 🚀 Funcionalidades

- Geração aleatória de um número secreto.
- Validação das tentativas do usuário.
- Dicas informando se o número é maior ou menor que o palpite.
- Mensagem de sucesso ao acertar o número.
- Opção de iniciar uma nova partida.

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## 🎯 Objetivo do projeto

Este projeto foi desenvolvido com o objetivo de praticar conceitos fundamentais de JavaScript, como:

- Manipulação do DOM
- Eventos
- Condicionais
- Laços de repetição
- Funções
- Geração de números aleatórios
- Lógica de programação

## ▶️ Como executar

1. Clone este repositório.
2. Abra o arquivo `index.html` em seu navegador.
3. Tente descobrir o número secreto!

# 🔢 Secret Number

**Secret Number** is a simple JavaScript project consisting of a number guessing game. The goal is to guess the secret number randomly generated by the system, using the hints provided after each attempt.

## 🚀 Features

- Random secret number generation.
- User guess validation.
- Hints indicating whether the secret number is higher or lower.
- Success message when the correct number is guessed.
- Option to start a new game.

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 🎯 Project Purpose

This project was created to practice fundamental JavaScript concepts, including:

- DOM manipulation
- Event handling
- Conditional statements
- Loops
- Functions
- Random number generation
- Programming logic

## ▶️ Getting Started

1. Clone this repository.
2. Open the `index.html` file in your browser.
3. Try to guess the secret number!
